How to Install FetchFast

  1. Click the Download button to get the FetchFast ZIP file.
  2. Unzip the downloaded file to a folder on your computer.
  3. Open Chrome and go to the Extensions page (chrome://extensions/), or select "Extensions" from the Chrome menu in the top right corner of the browser.
  4. Enable "Developer mode" by toggling the switch in the top right corner of the Extensions page.
  5. Click the "Load unpacked" button and select the folder where you unzipped FetchFast.
  6. FetchFast is now installed and ready to use! You'll see the FetchFast icon in your browser's extension area.
